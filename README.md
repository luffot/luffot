## 🎯 项目愿景

Luffot 是一款以用户为中心的**响应式智能桌面陪伴 AI**，旨在成为您在数字世界中的"第二大脑"。它通过持续学习与环境交互，帮助您高效管理注意力、提升生产力，并在关键时刻提供及时、可靠的决策支持。

不同于传统依赖用户主动发起请求的命令式 AI 助手，Luffot 采用**被动感知与事件驱动**的响应式架构，实现智能化、无感化的主动服务。

## ✨ 核心特性

### 🤖 多智能体协同架构

Luffot 采用**分布式多智能体架构**，由多个专业化、可独立运行的智能体组成，各司其职，通过统一事件总线进行通信与协作。

| 智能体 | 职责 | 核心能力 |
|--------|------|----------|
| **系统管家** | 监控系统资源与运行状态 | CPU/内存/网络监控、进程异常检测、系统健康度报告 |
| **用户习惯记录员** | 追踪分析用户软件使用模式 | 高频应用识别、使用时长统计、动态用户画像构建 |
| **应用秘书** | 监听应用内关键事件 | 钉钉/微信/QQ 消息监听、事件上下文提取、结构化上报 |
| **摄像头巡查员** | 环境感知与安全监控 | 人员进入检测、物品移动识别、隐私优先本地处理 |
| **语音助手** | 接收语音指令并执行 | 语音唤醒、意图识别、智能对话 |
| **Luffot 秘书长** | 中枢协调与决策 | 事件聚合、优先级排序、语义融合、主动信息推送 |

### 🎮 沉浸式桌面宠物交互

- **智能状态感知**：根据用户行为自动切换状态（摸鱼/敲键盘/思考/说话/告警）
- **AI 对话气泡**：支持流式对话、打字机效果、Markdown 富文本渲染
- **拖拽交互**：支持鼠标拖拽移动、单击唤出对话窗口
- **多皮肤系统**：支持矢量皮肤、图片皮肤、Lua 脚本皮肤（LÖVE2D）
- **弹幕通知**：消息以弹幕形式飘过屏幕，支持关键词高亮与过滤

### 📱 应用消息智能监听

- **多平台支持**：钉钉、微信、QQ、阿里钉等主流应用
- **双模式监听**：
  - Accessibility 模式：通过系统 API 快速读取窗口文本
  - VLModel 模式：截图后调用视觉模型识别，准确率更高
- **智能去重**：基于消息内容哈希自动去重
- **会话识别**：自动识别聊天会话，构建消息上下文

### 🧠 AI 智能分析

- **多 Provider 支持**：OpenAI、阿里云百炼、DashScope、本地 CoPaw Agent
- **流式对话**：支持 SSE 流式响应，实时展示 AI 思考过程
- **视觉理解**：支持图片分析（摄像头监控、截图识别）
- **智能摘要**：紧急消息自动摘要，高优先级内容主动推送
- **用户画像**：基于消息内容和使用习惯构建个人画像，提供个性化服务

### 🔔 智能告警系统

- **关键词告警**：支持自定义告警关键词，命中时触发桌宠告警状态
- **屏幕边缘警示**：告警时屏幕边缘红色渐变遮罩提醒
- **摄像头背后守卫**：检测身后人员，保护隐私安全
- **过滤机制**：支持过滤关键词，避免误报

### 📊 Web 管理界面

- **消息中心**：历史消息浏览、搜索、统计
- **应用管理**：添加/编辑/删除监听应用，配置解析规则
- **AI 配置**：多 Provider 管理、模型切换、参数调整
- **皮肤管理**：皮肤切换、自定义皮肤导入
- **定时任务**：Cron 表达式配置、内置任务/Python 脚本支持
- **提示词管理**：系统提示词动态编辑

## 🚀 快速开始

### 环境要求

- **操作系统**: macOS 15+ (当前主要支持 macOS)
- **Go 版本**: 1.24+
- **辅助功能权限**: 需要授予辅助功能权限以监听应用消息

### 安装运行

```bash
# 克隆仓库
git clone https://github.com/luffot/luffot.git
cd luffot

# 安装依赖
go mod tidy

# 运行（首次运行会自动创建默认配置）
go run ./cmd/luffot
```

### 配置文件

首次运行后，配置文件将创建于 `~/.luffot/config.yaml`：

```yaml
ai:
  enabled: true
  default_provider: "default"
  max_context_rounds: 10
  providers:
    - name: "default"
      provider: "bailian"  # 或 openai/dashscope/copaw
      api_key: "your-api-key-here"
      model: "qwen-plus"

web:
  enabled: true
  host: "127.0.0.1"
  port: 8765

apps:
  - name: "dingtalk"
    process_name: "DingTalk"
    display_name: "钉钉"
    enabled: true
```

详细配置说明请参考 [HowToStart.md](./HowToStart.md)。

## 📸 功能展示

### 桌面宠物交互
- 智能感知用户键盘输入，自动切换"敲键盘"状态
- 单击宠物唤出 AI 对话窗口，支持自然语言交互
- 拖拽移动宠物到屏幕任意位置

### 消息弹幕
- 应用消息以弹幕形式实时飘过屏幕
- 支持关键词高亮（如紧急消息金色显示）
- 支持关键词过滤（如过滤群消息）

### 智能告警
- 检测到紧急关键词时，桌宠进入"告警"状态
- 屏幕边缘红色渐变遮罩提醒
- 摄像头检测到身后有人时主动告警

### Web 管理
- 访问 http://127.0.0.1:8765 打开管理界面
- 查看历史消息、统计数据
- 配置 AI Provider、管理皮肤

## 📝 设计理念

1. **去中心化感知，中心化决策**  
   各智能体独立感知局部信息，由 Luffot 秘书长进行全局整合与策略输出。

2. **隐私优先**  
   所有敏感数据（摄像头画面、语音录音、消息内容）默认在本地处理，仅结构化元数据上送。

3. **可扩展性强**  
   新增应用或设备只需部署对应智能体，无需重构主系统。

4. **用户主权**  
   Luffot 秘书长是用户私有的数字秘书，所有服务完全围绕用户需求展开。
